Detailed Description

This script implements a Lua API framework for your Unity scene.

Includes methods to execute custom Lua scripts, expose game objects to Lua, and register and invoke events. Inherits from MonoBehaviour. Intended use is to add to an object in the scene and set up for use.

This class registers and invokes the SceneLoaded event when the scene is loaded.
