Detailed Description

Class structure which establishes a Lua object framework.

You may use this class as a base for your game object wrappers. All public members of derived classes will be exposed to Lua.

◆ ID

string uLua.LuaClass.ID
get

Used to access the unique ID of the Lua object.

The unique ID for a Lua object is defined as: 'Context.Name'. If an object's context is null, its unique ID is simply its name.

Implements uLua.IHasLuaIndexer.
